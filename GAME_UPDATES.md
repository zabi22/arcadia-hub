## 🎯 HOW TO PLAY

### Snake:
1. Choose mode (Classic/Speed/Survival)
2. Click Start (overlay will fade out)
3. Use arrow keys or WASD
4. Eat food, avoid walls/self
5. Collect power-ups from shop

### Memory Match:
1. Choose difficulty
2. Click cards to flip
3. Match pairs consecutively for combo bonus
4. Fewer moves + faster time = higher score

### Reaction Time:
1. Click Start
2. Wait for green
3. Click as fast as possible
4. Build streak for bonus points
5. Stay under 300ms to maintain streak
